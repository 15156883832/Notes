JAVA支持多继承吗？----java类是不支持多继承的，但是java接口支持多继承，一个接口A可以有多个父接口B,c是A的实现类，则c需要实现A和B接口中所有的方法

接口：是绝对抽象的，接口中所有的方法隐含的都是抽象的，申明的变量默认都是final的

抽象类：可以同时包含抽象方法和非抽象方法


接口和抽象类都不能被实列化

java中值传递一般是针对基本数据类型的，引用传递是针对实例对象的
